# 📚 OUSL Book Shop

A web-based online bookshop application developed as part of an academic assignment 
at The Open University of Sri Lanka (OUSL). The system allows users to browse, 
search, and purchase books through a simple and user-friendly interface, while 
storing book and order data in a backend database.

## 🌐 Live Demo
https://sahanwickramasingha.github.io/OUSLBookShop/

## ✨ Features
- Browse and view available books
- Product listing with images and details
- Simple, responsive front-end design
- Database-backed storage for book records
- Order/checkout process

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS
- **Backend:** PHP (htdocs)
- **Database:** MySQL
